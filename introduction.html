Introduction

Ce site a été crée lors de la SAé-1.4.
Cette SAé a pour but de créer un site web, dans lequel je vais me présenter, dont les contraintes sont les suivantes :
  • le site Web doit avoir au moins 3 pages
  • toutes les pages sont en français, excepté une page AUTRE que la page d'accueil, qui est en anglais
  • doit contenir toutes les balises du slide 31 du cours ("Main skills to have"), avec au moins 3 sélecteurs et 3 propriétés différentes de CSS
  • les images doivent être légendées (+ spécifier la source d'où l'image a été prise) ; une image que vous avez créée vous-même sur ordinateur sera un plus
  • toutes les pages de votre site et les CSS doivent passer la validation sur le site W3C (en particulier, la balise center est obsolète, donc ne pas l'utiliser).
Tout d'abord, que signifie le sigle SAé ? le sigle SAé signifie "Situation d'Apprentissage et d'évaluation". Chaque SAé a pour but de mettre en avant notre autonomie et notre créativité.
De plus, dans cette SAé, nous devrons également poster une vidéo de présentation de nous même, une carte mentale, des traces de notre travail, notre portfolio
